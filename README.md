# csharp-journey

# 30-Day C# Learning Roadmap

## Week 1: Foundations (Days 1-7)

### Day 1: Setup & Hello World
- Install Visual Studio Community or VS Code with C# extension
- Create your first console application
- Understand basic program structure (namespace, class, Main method)
- *Practice:* Write 5 different "Hello World" variations with different messages

### Day 2: Variables & Data Types
- Learn primitive types (int, double, float, bool, char, string)
- Variable declaration and initialization
- Type conversion and casting
- *Practice:* Create a program that converts temperature (Celsius to Fahrenheit)

### Day 3: Operators & Expressions
- Arithmetic operators (+, -, *, /, %)
- Comparison operators (==, !=, <, >, <=, >=)
- Logical operators (&&, ||, !)
- *Practice:* Build a simple calculator (add, subtract, multiply, divide)

### Day 4: Control Flow - Conditionals
- if, else if, else statements
- switch statements
- Ternary operator
- *Practice:* Create a grade calculator (input score, output letter grade)

### Day 5: Loops
- for loops
- while loops
- do-while loops
- break and continue
- *Practice:* Create a multiplication table generator

### Day 6: Arrays
- Declaring and initializing arrays
- Accessing array elements
- Array methods (Length, Sort, Reverse)
- Multi-dimensional arrays
- *Practice:* Store and display student grades, find average

### Day 7: Review & Mini Project
- Review everything from Week 1
- *Project:* Build a simple number guessing game
  - Generate random number
  - Take user input
  - Give hints (higher/lower)
  - Count attempts

---

## Week 2: Methods & OOP Basics (Days 8-14)

### Day 8: Methods/Functions
- Method declaration and syntax
- Parameters and arguments
- Return values
- Method overloading
- *Practice:* Create utility methods (FindMax, IsEven, ReverseString)

### Day 9: Classes & Objects - Part 1
- Understanding classes and objects
- Fields and methods
- Creating instances
- The this keyword
- *Practice:* Create a Person class with name, age, and Introduce() method

### Day 10: Constructors
- Default constructors
- Parameterized constructors
- Constructor overloading
- *Practice:* Extend Person class with multiple constructors

### Day 11: Encapsulation & Properties
- Access modifiers (public, private, protected)
- Properties (get/set)
- Auto-implemented properties
- *Practice:* Create a BankAccount class with balance, deposit, withdraw

### Day 12: Static Members
- Static fields and methods
- Static classes
- When to use static
- *Practice:* Create a MathHelper class with static utility methods

### Day 13: Inheritance
- Base and derived classes
- The base keyword
- Method overriding (virtual/override)
- *Practice:* Create Animal base class, then Dog and Cat subclasses

### Day 14: Review & Mini Project
- Review Week 2 concepts
- *Project:* Build a simple library system
  - Book class (title, author, ISBN)
  - Library class (add books, search, display)
  - Use inheritance for different book types

---

## Week 4: LINQ, Events & File I/O (Days 22-28)

### Day 22: LINQ Basics - Part 1
- Introduction to LINQ
- Where, Select, OrderBy
- First, FirstOrDefault
- *Practice:* Query and filter student lists

### Day 23: LINQ Basics - Part 2
- GroupBy, Sum, Count, Average
- Any, All
- Joining collections
- *Practice:* Analyze sales data with LINQ queries

### Day 24: Delegates
- Understanding delegates
- Action and Func delegates
- Predicate delegates
- *Practice:* Create a filter method using delegates

### Day 25: Events
- Event declaration and subscription
- EventHandler
- Custom events
- *Practice:* Create a simple notification system

### Day 26: File I/O
- Reading from files (File.ReadAllText, ReadAllLines)
- Writing to files (File.WriteAllText, WriteAllLines)

- *Practice:* Save and load todo list from file

### Day 27: Generics
- Generic classes and methods
- Type constraints
- Benefits of generics
- *Practice:* Create a generic Stack<T> class

- *Start Capstone:* Task Management Application
  - User authentication (simple)
  - CRUD operations for tasks
  - File persistence
  - Priority levels using enums

## Daily Study Tips

- *Study Time:* 2-3 hours per day minimum
- *Practice:* Write code every single day
- *Resources:*
  - Microsoft C# Documentation
  - YouTube: Mosh Hamedani, Tim Corey
  - Practice: HackerRank, LeetCode (easy problems)
- *Don't skip days* - consistency is key
- *Experiment* - modify examples and break things to learn
- *Take notes* - write down key concepts
